
 * 当路径有含义时候，其路径结尾含义定义如下

| X取值     | Y取值               | 类型 | 备注 |
| ----------- | ------------------ | -------- | ------------ |
| X % 4 == 0      |  Y % 4 == 0               | 正常清扫弓字轨迹 |  |
| X % 4 == 0      |  Y % 4 == 1               | 回充轨迹 |  |
| X % 4 == 0      |  Y % 4 == 2               | 遥控轨迹 |  |
| X % 4 == 0      |  Y % 4 == 3               | 用户定义区域的转场路径 |  |
| X % 4 == 1      |  Y % 4 == 0               | 清扫转场轨迹 |  |

>> 提示 : (X % 2 == 0) 等效于 ((X % 4 == 0) || (X % 4 == 2))

## Install

```sh
$ npm install @tuya/tuya-panel-robot-sdk
// or
$ yarn add @tuya/tuya-panel-robot-sdk
```

## Usage

```js
import { RCTGyroMap } from '@tuya/tuya-panel-robot-sdk';

...
```

## Install

```sh
$ npm install @tuya/tuya-panel-ipc-sdk
// or
$ yarn add @tuya/tuya-panel-ipc-sdk
```

## Usage

```js
import { TYIpcPlayer, TYIpcNative, TYIpcTopBar } from '@tuya/tuya-panel-ipc-sdk';

...
```

## 安装

```sh
$ npm install @tuya/tuya-panel-sensing-sdk
// or
$ yarn add @tuya/tuya-panel-sensing-sdk
```

## 使用

```js
import { useDpState } from '@tuya/tuya-panel-sensing-sdk';
```

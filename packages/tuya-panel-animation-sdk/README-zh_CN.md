## 安装

```sh
$ npm install @tuya/tuya-panel-animation-sdk
// or
$ yarn add @tuya/tuya-panel-animation-sdk
```

## 使用

```sh
import { Diffusion } from '@tuya/tuya-panel-animation-sdk'
```

## 简介

_负责人: Ivvan（黄婷婷）

`ScaleCarousel` 是基于轮播图改造的轮播切换组件。

## 代码演示

### 基础使用

<Playground>
 {() => {
    const [state, set] = React.useState({ selectedIndex: 0 });
    const setState = value => set({ ...state, ...value });
    return (
      <ScaleCarousel  
        selectedIndex={selectedIndex}
        carouselChange={index => {
          setState({ selectedIndex: index });
        }}
      />
    )
 }}
</Playground>
